### 三、使用分支

```js
1、git branch -a   //所有分支展示
  ==》remotes/origin/HEAD -> origin/master
  ==》remotes/origin/develop
  ==》remotes/origin/master

2、 git checkout -b develop origin/develop  //切换到那个分支

3、git branch //查看当前所在分支
```
